GITHUB : RÉPERTOIRES PERSONNELS

QU’EST CE QUE GITHUB ?

GitHub est un service en ligne qui permet d’héberger des répertoires de code. C’est un outil gratuit pour héberger du code open source, et il propose également des plans payants pour les projets privés. C’est le numéro 1 mondial et il héberge plus de 10 millions de répertoires. Microsoft a annoncé le lundi 4 juin 2018 l’acquisition pour 7,5 milliards de dollars de la plate-forme GitHub, ce qui montre bien l’importance de cet acteur dans le domaine du développement numérique au niveau mondial.

L’UTILISATION DE GITHUB

Au cours de notre formation, nous avons été amené à créer un compte GitHub, afin d’utiliser sa version GitHub Desktop et Visual Studio Code. Tous deux nous ont permis de créer nos premières pages HTML, d’apprendre le CSS ainsi que de travailler en groupe sur le même projet. Très pratique GitHub permet donc de travailler à plusieurs sur les mêmes fichiers sans risquer de compromettre le projet en cours. Il contient toutes les sauvegardes du projet et permet ainsi d’effectuer des backups de sécurité et des sauvegardes avant test.
