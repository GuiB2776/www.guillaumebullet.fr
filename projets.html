Psych'Object

2ème Projet Étudiant


Le deuxième projet mené lors de notre 1ère année concernait la création et mise en place d’un site e-commerce, avec pour objectif de présenter des goodies sur le thème voulu. Mon groupe et moi-même avons donc choisi pour thème principal : les films d’horreurs cultes du cinéma, plus précisément la sous-catégorie des slashers/thrillers. Comprenant des oeuvres comme Shining, Ça, Saw, Chucky, Vendredi 13, etc… le site comporte une partie blog dans laquelle les utilisateurs peuvent retrouver des anecdotes, histoires, informations sur les lieux de tournages ainsi que leurs synopsis.

GitHub

Mes répertoires GitHub


GitHub est un service en ligne qui permet d’héberger des répertoires de code. C’est un outil gratuit pour héberger du code open source et il propose également des plans payants pour les projets privés. C’est le numéro 1 mondial et il héberge plus de 10 millions de répertoires. Microsoft a annoncé le lundi 4 juin 2018 l’acquisition pour 7,5 milliards de dollars de la plate-forme GitHub, ce qui montre bien l’importance de cet acteur dans le domaine du développement numérique au niveau mondial.
